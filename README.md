Set of console mini-games

## Install

```bash
$ npm install -g brain-games-gudzii-ov
```

## Play games

### Is number even game
```bash
$ brain-even
```

### Calculator game
```bash
$ brain-calc
```

### Greatest common divisor game
```bash
$ brain-gcd
```

### Balance game
```bash
$ brain-balance
```

### Arithmetic progression
```bash
$ brain-progression
```

### Is number prime game
```bash
$ brain-prime
```
